# Terminbuchungs-Webseite

Dies ist eine einfache, leichtgewichtige Webseite zur Online-Terminbuchung. Sie integriert das [Timify](https://www.timify.com/) Buchungs-Widget nahtlos über einen Button.

## 🌐 Live-Ansicht
Die Webseite wird automatisch über GitHub Pages gehostet und ist hier erreichbar:
👉 **[DEIN-LINK-HIER, z. B. https://dein-benutzername.github.io/dein-repo/]**

## ⚙️ Funktionen
* **Responsive Design:** Sieht auf Smartphones, Tablets und Desktop-Computern gut aus.
* **Timify Integration:** Das Buchungs-Widget wird sicher geladen und öffnet sich direkt auf der Seite.
* **Schnelle Ladezeit:** Das externe Skript wird asynchron (`async`) geladen, damit die Webseite nicht blockiert wird.

## 🛠️ Anpassungen vornehmen
Die gesamte Webseite befindet sich in der Datei `index.html`. 
1. Klicke auf die Datei `index.html` hier in GitHub.
2. Klicke oben rechts auf das kleine **Stift-Symbol** (Edit this file).
3. Du kannst Texte (wie "Willkommen auf unserer Webseite") oder Farben im `<style>`-Bereich direkt anpassen.
4. Klicke oben rechts auf **Commit changes**, um zu speichern.

*Hinweis: Nachdem du Änderungen gespeichert hast, dauert es meist 1-2 Minuten, bis GitHub Pages die Live-Webseite im Hintergrund aktualisiert hat.*
